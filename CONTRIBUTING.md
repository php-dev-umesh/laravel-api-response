# Contributing

Contributions are welcome and will be fully credited.

## Pull Requests

- **Use the PSR-12 coding standard.**
- **Add tests** for any new features or bug fixes.
- **Document any changes** in the README and CHANGELOG.
- **Create feature branches** — don't commit directly to main.

## Running Tests

```bash
composer test
```

## Reporting Issues

Please report issues via the [GitHub issue tracker](https://github.com/php-dev-umesh/laravel-api-response/issues).
